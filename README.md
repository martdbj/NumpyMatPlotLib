## Como ejecutar la aplicación
Clona el repositorio y escribe en poweshell:
```bash
git clone https://github.com/martdbj/NumpyMatPlotLib.git
cd .\NumpyMatPlotLib\TrabajoFinal\
```
Ejecuta este comando para inicializar la aplicación:
```bash
py .\Main.py
```
